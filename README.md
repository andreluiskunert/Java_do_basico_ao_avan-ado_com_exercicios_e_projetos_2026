# ☕ Curso Completo de Java

Bem-vindo ao curso mais completo de **Java**, indo do básico ao avançado!

## 📚 Sobre o Curso
Aprenda uma das linguagens mais utilizadas do mundo com aulas práticas, exercícios e projetos reais com regras de negócio.

## 🚀 Conteúdos Abordados
- Variáveis e tipos de dados  
- Estruturas condicionais (`if`, `else`, `switch`)  
- Estruturas de repetição (`for`, `while`, `do while`)  
- Arrays e Funções  
- Orientação a Objetos (POO)  

## 🔥 Conteúdos Avançados
- Tratamento de erros  
- Manipulação de arquivos  
- Generics  
- Collections (`List`, `Set`, `Map`)  
- Expressões regulares  
- Annotations  
- JavaFX (aplicações desktop)  

## 🧠 Para Quem é?
Não é necessário conhecimento prévio. Ideal para iniciantes e para quem deseja se preparar para o mercado como desenvolvedor.

## 🛠️ Diferenciais
- Projetos práticos completos  
- Suporte para dúvidas  
- Atualizações constantes  
- Acesso vitalício  
- Certificado de conclusão  
- Garantia de reembolso  

## 🎯 Objetivo
Preparar você para o mercado de trabalho dominando Java e conceitos fundamentais de programação.
